С камушка на камушек порхал воробышек, а на террасе, искусно задрапированной гобеленами с дефензивой кронштадтского инфантерийского батальона, под искусственным абажуром, закамуфлированным под марокканский минарет, веснушчатая свояченица вдовствующего протоиерея Агриппина Саввична потчевала коллежского асессора, околоточного надзирателя и индифферентного ловеласа Фаддея Аполлинарьевича винегретом со снетками.
